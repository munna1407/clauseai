# ClauseAI

AI-powered contract analysis system.

## Features
- Upload contracts
- AI clause analysis
- Vector search using Pinecone
- Gemini embedding integration

## Tech Stack
- Python
- FastAPI
- Pinecone
- Google Gemini

## Setup

1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt
3. Add your .env file
4. Run the app
